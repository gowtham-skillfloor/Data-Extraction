# Data Extraction Project


**How to contribute to this project?**
---


Each contributor should focus on writing one function at a time in one file only. For example, if a contributor wants to extract email,
they should create a file called `email.py` and in that file they should write a function with name `email_extractor`.

The function should read a file, go through the line by one-by-one an extract emails and store it in a list and return the list object.

Please test the function locally before creating a Pull Request (PR).


**Hint**: Only use Python programming language.
